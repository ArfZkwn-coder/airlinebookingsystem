## 🤖 What is Mimi?

**Mimi** is a witty, elite senior developer agent built into VS Code Copilot. She's specialized for teams building **Next.js + TypeScript + Prisma** applications with strict quality standards.

### Mimi's Core Features
- ✅ **Strict Type Safety** — Zero tolerance for `any` types
- ✅ **Tailwind CSS Exclusively** — No other styling frameworks
- ✅ **Mandatory Testing** — Jest + React Testing Library for all UI
- ✅ **Opinionated Tech Stack** — Next.js, Prisma, Zustand, React Query
- ✅ **Distinctive Personality** — Formal, protective, witty senior dev persona
- ✅ **Code Quality Enforcement** — Components under 100 lines, kebab-case files, PascalCase components

## 🚀 Quick Start

### 1. Copy Mimi to Your Workspace
```bash
cp Mimi.md /path/to/your/project/
```

### 2. Reload VS Code
```
Ctrl+R (Windows/Linux) or Cmd+Shift+R (Mac)
```

### 3. Open Copilot Chat
```
Ctrl+I
```

### 4. Start Using Mimi
Ask Mimi to scaffold a component, review code, or suggest improvements:
```
Mimi, scaffold a form component with validation 
and TypeScript types. Include full test coverage.
```
